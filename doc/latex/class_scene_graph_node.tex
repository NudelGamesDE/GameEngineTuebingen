\subsection{Detailed Description}
A scene graph node class. 

This class implements a scene graph node. By multiplying a parent\textquotesingle{}s transformation matrix to a node\textquotesingle{}s own, the node is affected by the parent. Every node has its own renderer which can later be passed to the scene.
