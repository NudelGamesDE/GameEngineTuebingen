Detailed Description

A renderer class.

The renderer class is a container class containing a mesh, a material, a transform and eventually a second transform aalong with a rigid body. By calling the draw function of a renderer the material used takes care of most shader operations while the mesh actually draws the vertices. A second transform mostly only is applied when the renderer is part of a scenegraph. The rigid body only needs to be implemented in case the drawn object should be influencable by forces and velocity.

Draw renderer.

This function draws the Mesh with the corresponding Material. It calls Material's Use function to set up the shader correctly and then Mesh's draw function to draw the vertices. If a RigidBody exists, its traits are applied to the transform.
