Detailed Description

A mesh class.

This class contains the vertices, uvs and normals of a mesh. This class is responsible for setting up buffers for and drawing vertices. It also checks for a hit while raycasting.

Constructor for a Mesh instance.

Generates buffers for data and for indices. All achieved data from parameters is written in a vertex buffer object, afterwards they get indexed. In the end a vertex array object is created and vertex attribute pointers are created.

void Mesh::Draw ( )

Draw the mesh.

This function draws the mesh by binding both vao and the indexed vbo and enabling the vertex arrays before calling glDrawElements. Note that all shader related operations are executed in the Material class.
